# Shopee Manager API

Dashboard para gerenciar produtos e pedidos da Shopee via API oficial.

## Como usar

1. Clone o repositório
2. Execute `npm install`
3. Configure suas credenciais da Shopee no código
4. Execute `node app.js`
5. Acesse `http://localhost:3000/dashboard`

## Funcionalidades

- ✅ Conexão com API da Shopee
- ✅ Dashboard de produtos
- ✅ Dashboard de pedidos  
- ✅ Análise competitiva
- ✅ Exportação de dados

## Tecnologias

- Node.js
- Express
- Shopee API v2
- HTML/CSS/JavaScript
